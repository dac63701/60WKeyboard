# Custom Keyboard Project
A wireless, 60% layout, hot swappable, custom keyboard designed with an nRF52840 in Kicad

## Criteria
- Wireless
- 60% Layout
- RGB Lighting
- Good Battery Life
- Cherry MX Switch Form Factor
- Hotswap Switches
